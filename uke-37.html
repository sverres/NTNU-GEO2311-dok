# GEO2311 - Høst 2016 - uke 37 ## Målsetting - Få erfaring med å lage egne websider med aktiv bruk av HTML og CSS - Lære mer om norske datasett - Lære om javascript ## Aktiviteter Denne uka skal vi sette sammen og vise det vi har lært om HTML og CSS ved å lage en egen hjemmeside. De to oppgitte kapitlene i læreboka er viktig for å forstå hvordan vi kan bygge opp en egen layout. Vi skal også starte litt forsiktig med javascript. - lese kapittel 8 i læreboka, bl.a. id og class - lese kapittel 15 i læreboka, om layout - hjemmeside-oppgave (kommer) - webkart-oppgave (kommer) - lese om javascript og programmering - se ressurser - lese om norske datasett (kommer) ## Forelesning/fellesaktivitet på torsdag - gruppearbeid om HTML- og CSS-teori (start 9:15) - arbeid med hjemmeside-oppgave - arbeid med webkart-oppgave - forelesning om javascript og programmering (13:15) - presentasjon av oblig 1 ## Om aktiviteter i uke 38 Onsdag 21. - torsdag 22. september er det Idelab-24.
